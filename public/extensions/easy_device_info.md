# EasyDeviceInfo [Beta / Free/OS]

EasyDeviceInfo is a simple extension to **read device information** such as system, hardware, and battery details on Android devices.  
It focuses on providing blocks that return information only; it does not add UI components by itself and some blocks may require specific permissions.

## New Update V(0.3)

- New blocks have been added to extend the information you can get from the device.
- You must now **initialize** the extension using the `Initialize` block before calling other blocks.
- A new property block **NotFoundVal** lets you set the value returned when a result is not found, invalid, or null.
