# Device Year [Discontinued]

> The maximum year returned is **2016** because the underlying library used by this extension has been archived.

Device Year is a simple Android/Kodular extension that analyzes a device’s hardware specifications and estimates which year the device would be considered “high end”, based on Facebook’s `device-year-class` library.
