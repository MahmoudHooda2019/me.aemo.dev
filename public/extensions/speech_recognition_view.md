# SpeechRecognitionView

SpeechRecognitionView is a UI extension that displays a **Google Now–style** animation while recording speech.  
It works together with a speech recognizer to provide a visual indication that audio is being captured.

## Notes

- Requires the `android.permission.RECORD_AUDIO` permission at runtime.  
- Needs an active internet connection to work correctly, even though the extension itself does not request internet permission directly.
